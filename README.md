# Customer Shopping Data Analysis

## Project Overview

This project analyzes **customer shopping behavior** using data analysis techniques in Python.
The objective is to identify purchasing patterns, customer preferences, and sales trends through data visualization and statistical analysis.

The analysis helps businesses understand customer behavior and improve marketing and sales strategies.

---

## Technologies Used

* Python
* NumPy
* Pandas
* Matplotlib
* Seaborn

---

## Dataset

The dataset contains information about customer shopping activities, including:

* Customer Age
* Product Category
* Quantity Purchased
* Price
* Payment Method
* Shopping Mall / Location

---

## Project Features

The project includes the following data analysis tasks:

* Data cleaning and preprocessing
* Exploratory Data Analysis (EDA)
* Distribution analysis of customer age and purchases
* Correlation analysis between variables
* Category-wise purchase trends
* Payment method analysis
* Visualization of customer shopping patterns

---

## Data Visualizations

The project uses **Matplotlib and Seaborn** to generate visual insights such as:

* Purchase distribution charts
* Category-wise sales comparison
* Payment method usage analysis
* Customer demographic patterns

---

## Project Goal

The main goal of this project is to demonstrate how **Python data science libraries** can be used to analyze customer shopping data and generate meaningful insights.
